## Run Locally

Clone the project

```bash
  git clone https://github.com/joel-elunda/emirates
```

Go to the project directory

```bash
  cd emirates
```

Install dependencies

```bash
  pip install widget_tweaks
  pip install import_export
```

Start the server

```bash
  manage.py runserver
```
